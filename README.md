# JNNavigation

主要功能是简化导航栏样式方面的开发工作，以枚举方式快速定义导航栏的样式  <br/> 
可以自定义增加导航栏的样式FNNavationBarType  <br/> 
支持ios7以上系统，对ios10兼容  <br/> 
简化导航栏item的添加方法 <br/> 
---
优化导航栏样式模块  <br/>  
1、导航栏颜色 <br/> 
2、导航栏上面的字体颜色  <br/> 
3、导航栏的透明度 <br/> 
4、导航栏下面的分割线  <br/> 
5、导航栏过度动画优化 <br/>
