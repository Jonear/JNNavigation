# JNNavigation
主要功能是简化导航栏样式方面的开发工作，以枚举方式快速定义导航栏的样式
可以自定义增加导航栏的样式FNNavationBarType
支持ios7以上系统，对ios10兼容
简化导航栏item的添加方法

优化导航栏样式模块
1、导航栏颜色
2、导航栏上面的字体颜色
3、导航栏的透明度
4、导航栏下面的分割线
5、导航栏过度动画优化
